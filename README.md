# TODO APP

# Setup

- yarn install
- yarn start

# Features

- Add new task
- Mark task as completed / uncompleted
- Listing completed / uncompleted
- Filters
- Search task
- Edit task
- Delete task
- Empty state

# Technologies

- React
- HTML
- SCSS
